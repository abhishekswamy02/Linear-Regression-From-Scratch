##  Project Overview

This project predicts house prices using manually implemented Linear Regression. Instead of relying on prebuilt ML libraries, the algorithm is built step-by-step to demonstrate how predictions, loss calculation, and optimization work internally.

## Project Structure

- Dataset loading and preprocessing
- Exploratory Data Analysis
- Feature scaling
- Manual implementation of Linear Regression
- Loss calculation (MSE)
